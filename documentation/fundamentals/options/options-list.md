# Options list

{% hint style="success" %}
Most of the following defaults can be overwritten by [custom defaults](custom-defaults.md).
{% endhint %}

{% hint style="warning" %}
Defaults marked with _**colored bold italic**_ can’t be overwritten by custom defaults.
{% endhint %}
